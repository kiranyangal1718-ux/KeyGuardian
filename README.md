# 🛡️ KeyGuardian
**KeyGuardian** is a high-precision password entropy analyzer. It doesn't just check length; it calculates the mathematical "bits of entropy" to tell you exactly how hard your password is to crack.

## 🚀 Features
* **Live Circular Scanner:** Visual feedback on your security level.
* **Brute-Force Estimator:** Shows crack time in seconds, minutes, or centuries.
* **100% Privacy:** Your password never leaves your browser. No servers, no logs.

## 📊 How it works
KeyGuardian uses the entropy formula:  
**Entropy Bits = Password Length × log2(Character Pool Size)**
